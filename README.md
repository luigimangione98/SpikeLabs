## Getting Started

1. Clone the repository
2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

4. Open [http://localhost:3000](http://localhost:3000) to view it in your browser

## Tech Stack

- React 18
- TypeScript
- Tailwind CSS
- React Router v6
- Recharts for data visualization
